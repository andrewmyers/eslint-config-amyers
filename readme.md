# eslint-config-amyers

ESLint rules for my projects

## Installation
```bash
yarn add git://github.com/andrewmyers/eslint-config-amyers.git
```

## Usage

Add this to your package.json
```json
"eslintConfig": {
    "extends": "amyers"
}
```

Add this script to your package.json
```json
"lint": "eslint . --fix"
```

## Lint
```bash
yarn lint
```
